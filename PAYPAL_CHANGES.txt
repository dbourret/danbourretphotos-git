PayPal update contents

1. Adds a PayPal button below the existing Square card checkout.
2. Adds GET /api/config/paypal.
3. Adds POST /api/payments/paypal/create.
4. Adds POST /api/payments/paypal/:orderId/capture.

6. Reuses server-side pricing, S3 checks, WHCC fulfillment, calendar manual fulfillment, database storage, and order emails.
7. Updates Content Security Policy rules for PayPal.
8. Keeps the PayPal client secret on the server only.
